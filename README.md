# SimpleForm React Component

Quickly create simple React forms styled with Bootstrap 4

[Live Demo (contact form)](https://www.solarleague.org/about/contact/)

## Example

```js
<SimpleForm field1 field2 onSubmit={ (form) => console.log(form) } />
```

## Available Commands

In the project directory, you can run:

### `npm install`

Installs required node modules

### `npm run build`

Builds the package for production to the `dist` folder

### `npm test`

Runs tests
